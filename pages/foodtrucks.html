Project: Foodtrucks

Assignment

In a previous course I made a design for a food truck service. The assignment is to take this design and build it by using HTML, CSS and JavaScript. The focus of this course is not about the content of the website but to learn the semantics of HTML and new techniques in CSS. This is also the first course where I start to learn how to apply useful JavaScript in website.

Description

Are you looking to rent a food truck? Then you are in the right place! On this website you can search and filter until you find your favourite food truck. You can read more details by visiting the detail page and even take a peek into the menu card. If you have made your choice, contact the food truck and check for availability.

Highlights

Flexbox

During this course I got more information about flexbox and how it can be applied effectively. Using flexbox I start to get a feeling of its usefulness and this will probably change my coding future.

Responsive

By using flexbox and grid and using units like EM, REM and VW, VH the website has become responsive. By using these techniques the code I had to add in an media query was minimal.

JavaScript

In this website I applied some interactive elements by using JavaScript. I created a toggle for a CSS class so when you use the filter options the input fields appear. In the slider for the price it also gives you the feedback of the amount you are selecting.
