Project: ImageNaat Storygrafie

Assignment

This assignment is during my first coding course in this study. We have to create a website using semantic HTML and CSS. The subject was one of our choosing. The focus of this assignment was to get familiar with all the HTML and CSS elements and attributes/selectors.

Description

This is a website about my travels. You can read the stories of my travels with a lot of photographs to give an idea of how it was like.
