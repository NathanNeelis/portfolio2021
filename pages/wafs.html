Project: Movies Web app

Assignment

Building a single page web application (SPA) without frameworks. The purpose of this project is to take a step back to write a web app with vanilla JavaScript and HTML & CSS. As a subject I could choose any API to work with. In this project we focus on fetching data from an API and manipulating data so it is usable to render in the UI of an app. To understand the process of APIâ€™s and the interaction of users we are going to draw an actor diagram and an interaction diagram. During this project we will be doing code reviews with other students and refactoring our own code based on feedback.

Description

Do you also spend too much time looking for a decent movie to watch? With this application you can see right away what the top movies are at this very moment! Find a movie to your liking and make some popcorn instead of spending minutes to find a good title.

In this project I choose to work with TheMovieDB API and in particular the most popular movies at the moment and search based movie results and details.

Highlights

Search for a movie

In this web application you can search for a movie where you want some more information about. When you have found the movie you are looking for, you can visit its detail page where you can see a description of the movie and its rate.

Recently viewed movies section

Every time you view de detail page of a movie, it gets added to the recently added movies. Up to 10 movies in total! The latest in front of the row.
